
== Data Asal 

Basis data dengan _schema_ PBB pada sistem basis data Oracle.

== Kegiatan Manipulasi Yang Dilakukan

Melakukan pengambilan data pembayaran PBB-P2 untuk tanggal 4 Agustus 2021 sebagai bahan rekonsiliasi data dengan Bank Jateng.

== Hasil Manipulasi

Hasil manipulasi data dapat diakses pada alamat https://docs.google.com/spreadsheets/d/1s0VuV0PyD8po2m8DpFsPS2TLWEkxsrCUwZw1zYavOPQ/edit?usp=sharing

== _Script_ atau _command_ yang Digunakan Untuk Melakukan Manipulasi

----
select kd_propinsi||kd_dati2||kd_kecamatan||kd_kelurahan||kd_blok||no_urut||kd_jns_op nop, 
  thn_pajak_sppt, jml_sppt_yg_dibayar
from pembayaran_sppt
where tgl_pembayaran_sppt between to_date('040820210000','DDMMYYYYHH24MI') and to_date('050820210000','DDMMYYYYHH24MI')
order by kd_kecamatan, kd_kelurahan, kd_blok, no_urut, thn_pajak_sppt
----
